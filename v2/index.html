Service Health Configuration

Configure environments and credentials for the "Show Service Health" feature.

Enter environment names, one per line (e.g., qa, staging, prod).

Note: These credentials are different from Jenkins API credentials and are used specifically for health check endpoints.

Enter a regex pattern to shorten environment names. The pattern will be used to extract a portion of the Environment column value.

Leave empty to show full environment names. Example: "([^-]+)" extracts "prod" from "prod-us-east-1".
